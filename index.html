Jargo: Real-Time Stochastic Ridesharing Simulator

Visit on Github Submit Issue

Jargo is a real-time simulator that provides more realistic evaluation of ridesharing algorithms. It is implemented as a Java library. It provides inheritable base classes Client and Traffic for developing ridesharing algorithms and traffic functions. It also has an API for developing evaluators. To learn more about Jargo's data model, see Chapter 2 of the PDF user manual. A web version of this chapter is available here.

Jargo is written with literate programming.

The latest release is v0.9.0 (this is a pre-release). Click here to download. The user manual is available here.

Release distributions include the Jargo library, the PDF user manual, and scripts for launching the included command-line and graphical evaluators. If you have missing runtime dependencies, e.g. JavaFX for running the graphical evaluator, you will need to obtain those yourself.

A tutorial is included in Chapter 1 of the PDF user manual included in the release. A web version is available here.

The tutorial goes over how to install Jargo, start a simulation, and analyze results.
